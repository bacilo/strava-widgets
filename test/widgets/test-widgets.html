Strava Widgets - HTML Attribute Configuration

Pure HTML, Zero JavaScript Configuration

What you're seeing: All four Strava widgets configured purely via HTML data-attributes. No JavaScript initialization code required.

How it works: Each widget is a Custom Element (Web Component) that reads its configuration from HTML attributes like data-url, data-theme, data-accent, etc.

Use case: Perfect for static site generators (Jekyll, Hugo, Gatsby), CMSs (WordPress, Drupal), or any HTML-only environment.
